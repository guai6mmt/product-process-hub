# 销售准入文件管理系统 · 本机版 v1

为"产品销售准入的文件门禁"做的小系统。逻辑完全依据上级目录的
[`产品流程与文件管理_逻辑规格.md`](../产品流程与文件管理_逻辑规格.md)。

## 怎么运行
- **双击 `run.bat`** 即可（会自动打开浏览器 http://localhost:8765）。
- 或命令行：`python app.py`
- 前提：装了 Python 3（命令行运行 `python --version` 能看到版本即可）。**不需要安装任何第三方库**。
- 关闭：在黑色命令行窗口按 `Ctrl + C`。

## 它能做什么（飞书做不到的那几件）
1. **一键开产品**：填产品名 + 选机构 + 补充协议份数 + 有无海报 + 合同形态（合并/拆分），自动生成整张文件清单（消保/法审/用印要求已预填）。
2. **硬顺序守卫**：消保没通过，法审、用印改不了（系统直接拦截并提示）。
3. **退回自动派生新版本**：上传新版本会保留旧版本、并自动重置审查链重走。
4. **文件入库**：上传的文件存在本机 `data/files/`，可随时下载。
5. **驾驶舱 + 今日待办**：每个产品的备齐进度、每个文件的"下一步动作"自动算。

## 数据与备份
- 全部数据在本目录的 **`data/`** 文件夹：`app.db`（结构与状态）+ `files/`（文件本体）。
- **备份 = 复制整个 `data/` 文件夹**。换电脑把它拷过去即可。

## 规则要点（与规格一致）
- 代销协议（总对总 + 补充协议）：消保 + 法审 + 用印（双方）。
- 委托代销函：仅对方机构用印。
- 宣传材料（PPT/一页通/海报/双录话术/线上智能播报）：消保，且消保表单必传。
- 合同：合并 1 份 或 拆分 3 份（开产品时二选一）；消保 + 法审。
- 海报：随机构（"机构"页可设有无）。
